Core Features

📍

GPS Location

Captures coordinates at check-in and check-out to verify physical presence in classroom.

📷

QR Code Scanning

Camera-based QR scan confirms the correct class session. Scanned twice: in and out.

😊

Mood Tracking

5-point emoji mood scale captures student emotional state before class begins.

📝

Learning Reflection

Pre-class and post-class forms capture what students learned and their feedback.

💾

Local Storage

SQLite database stores all sessions offline. No internet required for core functionality.

📋

Session History

Expandable history screen shows all past sessions with full details and status.

User Journey

🔐 Before Class — Check-In
1Tap Check In button
2GPS location + timestamp auto-captured
3Scan instructor's QR code
4Fill previous & expected topics
5Select pre-class mood (1–5)
6Submit → saved to SQLite
🚪 After Class — Finish
1Tap Finish Class button
2Scan QR code again (confirms stayed)
3GPS location re-captured
4Write what was learned today
5Optional instructor feedback
6Submit → session marked complete
